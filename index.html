AlphaShapes

Driven by data ingested using Python. A financial data visualisation tool built in React, Alt.js and D3.js.

  • Project AlphaShapes has been lead by Benjamin King, and a cross-functional dev team of about 3-5 software engineers.
  • AlphaShapes is a financial data viz tool made to launch on Eikon.
  • It lets you select a financial criterion to rank and compare constituents of major stock indices like DJI, ASX, NYSE.
  • BE ingester built in Python - transformation of data. It periodically downloads and processes Thomson Reuters data into static JSON files to serve to the front-end.
  • FE - representation of data. Built in React, D3, AltJS.

    • Front-End: Persistence

    • One of first tasks implemented by pair programming with Anthony.
    • If you're a trader using AlphaShapes, there is a good chance you are you are going to be using near daily, and you are probably using AlphaShapes to keep track of a particular dataset.
    • It's important to make it easy for the user to re-open AlphaShapes with the same state it was last in - if they close the app looking at Total Revenue for Dow Jones, that's what they should see when they open it.
    • The solution is straight-forward: whenever the user makes some change, like the index they're viewing, the criteria they use, or the settings they use to view an AlphaShape, we serialize that state into a string and save that in local storage.
    • When the user starts up AlphaShapes, we've got up to date information on how they last left the app, and all we have to do is restore it.
